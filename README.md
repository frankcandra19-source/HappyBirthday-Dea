WEBSITE ULANG TAHUN – SIAP VERCEL

1. Ganti nama "Dea" di index.html sesuai nama pasangan.
2. Ganti semua file SVG di assets/images/ dengan foto sendiri.
   Pertahankan nama file agar tidak perlu mengubah kode.
3. Musik bersifat opsional. Masukkan lagu MP3 milik Anda ke:
   assets/audio/music.mp3
4. Upload seluruh isi folder ke GitHub lalu import ke Vercel,
   atau drag folder ke Vercel CLI.
5. Tidak memerlukan database atau proses build.

Catatan: Paket ini adalah implementasi orisinal yang terinspirasi oleh alur
website referensi, bukan salinan kode atau aset dari website tersebut.
